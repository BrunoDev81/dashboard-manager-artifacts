# Dashboard en blanco

Base corporativa para crear un dashboard dentro de Dashboard Manager.

## Archivos

- `dashboard.manifest.json`: contrato de ejecución y fuentes autorizadas.
- `index.html`, `style.css`, `app.js`: interfaz estática sin dependencias externas.

## Próximos pasos

1. Autorizá datasets o datos editables desde la administración del dashboard.
2. Actualizá el manifest solo con fuentes aprobadas.
3. Implementá la interfaz usando `DashboardManager` y validá el commit antes de importarlo.
